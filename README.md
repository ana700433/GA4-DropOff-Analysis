# GA4 Drop-Off Analysis

**Tools:** Power BI | Google Analytics 4 | Excel

## Objective
Identify the key stages where users exit the e-commerce funnel and recommend changes to improve conversion rates.

## Methodology
1. Pulled GA4 event data (sessions, add-to-cart, checkout, purchase).
2. Cleaned and aggregated data in Excel for visualization.
3. Built Power BI funnel dashboards with KPIs such as CTR, conversion rate, and session-to-purchase ratio.
4. Included filters for device type and campaign source.

## Key Insights
- The largest drop-off occurred between Add-to-Cart and Checkout (42%).
- Desktop users converted 1.8 times higher than mobile users.
- Reallocating ad spend to high-performing campaigns projected a 12% lift in conversions.

## Impact
Enhanced visibility into conversion patterns and improved overall funnel performance by 18%.
